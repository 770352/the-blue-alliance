TBATV GAE

Some things you may be interested in doing:

To process TaskQueues offline use the following command: python2.5 run_tasks.py -p port

Usage: run_tasks.py [OPTIONS] [queue-name]

Options: -h, --help show this help message and exit
-n NAME, --name=NAME name of the queue. default: default
-s HOST, --srv=HOST the host where the server runs. default: localhost
-p PORT, --port=PORT the port where the server listens. default: 8080
-l, --list Just list the first 25 tasks in the queue and exit

Setup

    You must do these things first.
  1. InstantiateUsfirstTeams, creates Team objects for all teams from USFIRST. Uses code borrowed from Patrick Fairbank's FRCLinks.
  2. InstantiateRegionalEvents, enqueues Tasks to create Event objects for regional events from USFIRST.

Updates

  • EnqueueGetUsfirstEvents, which sets up the TaskQueue to update details for all official Event objects.
  • EnqueueGetUsfirstMatchResults, which sets up the TaskQueue to get match results for all known Event objects, which will create or update Match objects, along with their affiliated Alliance objects. This will currently fail for Alliances which contain a Team that does not exist (I think, -greg 6/21/10).
  • EnqueueGetUsfirstTeams, which sets up the TaskQueue to read the USFIRST team details, updating Team objects.

View

  • Team List, attempts to list all teams. Eventually a public facing page.
  • Event List, attempts to list all events. Eventually a public facing page.
  • Match List, attempts to list all matches. Eventually a public facing page?
