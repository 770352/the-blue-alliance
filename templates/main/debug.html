The Blue Alliance Developer Page

Hello developer! Welcome to the debug page for The Blue Alliance. This page is full of useful pointers to help you locally develop TBA. It is different from the admin console, which is designed to help fix data in production.

To get started, you have to set up some initial test data for your local copy of TBA. We use paver scripts in pavement.py to set up this test data in your dev environment. easy_install paver if you don't have it.

Run these on the command line in the folder you have the-blue-alliance project in:

  1. paver dev_teams_setup
  2. paver dev_data_setup

This test data gives you the 2010 Championship Einstein Field as your main test data.

Prod Setup

If you're setting up a full copy of The Blue Alliance, these will help you create all the data you need.

  1. UsfirstEventsInstantiate, enqueues Tasks to create Event objects for regional events from USFIRST.
  2. Instantiate 2012 Teams: UsfirstTeamsInstantiate @0 @1000 @2000 @3000
  3. Instantiate 2011 Teams: UsfirstTeamsInstantiate @0 @1000 @2000
  4. Instantiate 2010 Teams: UsfirstTeamsInstantiate @0 @1000 @2000
  5. Instantiate 2009 Teams: UsfirstTeamsInstantiate @0 @1000 @2000
  6. Bulkload Events:
    appcfg.py upload_data --config_file=bulkloader.yaml --filename=test_data/events.csv --kind=Event --url=http://tbatv-prod.appspot.com/_ah/remote_api
  7. Bulkload Matches:
    appcfg.py upload_data --config_file=bulkloader.yaml --filename=test_data/matches.csv --kind=Match --url=http://tbatv-prod.appspot.com/_ah/remote_api
  8. Update EventTeams, to build all the event attendance information.

Updates

  1. UsfirstEventGetEnqueue, enqueue tasks to update official Events.
  2. UsfirstMatchesGetEnqueue, enqueue tasks to update match results for 2012 Events
  3. UsfirstTeamGetEnqueue @ 0, @ 1000, @ 2000, @ 3000, enqueue tasks to update Team details
  4. TBAVideosGet with an Event key to scrape http://videos.thebluealliance.net for TBAVideos.
  5. Update EventTeams, to build all the event attendance information.

Useful Other Commands

  • appcfg.py upload_data --config_file=bulkloader.yaml --filename=something.csv --kind=Something --url=http://something/_ah/remote_api
