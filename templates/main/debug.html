Dev Setup

Use paver to set up dev environment. easy_install paver if you don't have it.


paver dev_teams_setup
paver dev_data_setup

Look in pavement.py for the tasks in case they're failing.

Prod Setup

  1. Instantiate Teams From 2012: UsfirstTeamsInstantiate @0 @1000 @2000 @3000
  2. Instantiate Teams From 2011: UsfirstTeamsInstantiate @0 @1000 @2000
  3. Instantiate Teams From 2010: UsfirstTeamsInstantiate @0 @1000 @2000
  4. Instantiate Teams From 2009: UsfirstTeamsInstantiate @0 @1000 @2000
  5. Bulkload Events:
    appcfg.py upload_data --config_file=bulkloader.yaml --filename=test_data/events.csv --kind=Event --url=http://tbatv-prod.appspot.com/_ah/remote_api
  6. Bulkload Matches:
    appcfg.py upload_data --config_file=bulkloader.yaml --filename=test_data/matches.csv --kind=Match --url=http://tbatv-prod.appspot.com/_ah/remote_api
  7. Update EventTeams
  8. Update TBAVideos
    You must do these things first.
  1. UsfirstTeamsInstantiate, creates Team objects for all teams from USFIRST. Uses code borrowed from Patrick Fairbank's FRCLinks.
  2. UsfirstEventsInstantiate, enqueues Tasks to create Event objects for regional events from USFIRST.

Updates

  • UsfirstEventGetEnqueue, which sets up the TaskQueue to update details for all official Event objects.
  • UsfirstMatchesGetEnqueue, which sets up the TaskQueue to get match results for all known Event objects, which will create or update Match objects.
  • UsfirstTeamGetEnqueue @ 0 and UsfirstTeamGetEnqueue @ 1k, which set up the TaskQueue to read the USFIRST team details, updating Team objects.
  • TBAVideosGet with an Event key to scrape TBA.net for TBAVideos.

Useful Commands

  • appcfg.py upload_data --config_file=bulkloader.yaml --filename=something.csv --kind=Something --url=http://something/_ah/remote_api
  • To process TaskQueues offline use the following command: python2.5 run_tasks.py -p port
